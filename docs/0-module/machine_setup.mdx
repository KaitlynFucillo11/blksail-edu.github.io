## Visual Studio Code

### Extensions

Open Visual Studio Code and click on the Extensions icon in the sidebar.

#### [Live Share](https://marketplace.visualstudio.com/items?itemName=MS-vsliveshare.vsliveshare)

This extension allows you to share your code with others and collaborate in real
time.

#### [Remote Dev](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.vscode-remote-extensionpack)

This extension allows you to develop inside a container, on a remote machine, or
in the Windows Subsystem for Linux (WSL).

#### [Docker](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-docker)

This extension allows you to manage Docker containers and images from within
Visual Studio Code.

#### [Python](https://marketplace.visualstudio.com/items?itemName=ms-python.python)

This extension adds support for Python to Visual Studio Code.

#### [C/C++ Extension Pack](https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools-extension-pack)

This extension adds support for C/C++ to Visual Studio Code.

#### [Jupyter](https://marketplace.visualstudio.com/items?itemName=ms-toolsai.jupyter)

This extension adds support for Jupyter notebooks to Visual Studio Code.
